INSTALAÇÃO

Este zip tem apenas os ficheiros alterados.

1. Cola config.lua na raiz do recurso.
2. Substitui client.lua, server.lua e fxmanifest.lua.
3. Substitui html/index.html.
4. Substitui html/assets/css/style.css.
5. Substitui html/assets/js/init.js.

IMPORTANTE:
Não incluí fontes nem imagens neste zip.
Mantém a tua pasta:
html/assets/fonts/
html/assets/images/

Para usar aviação, coloca:
html/assets/images/aviation.png

TriggerServerEvent('jsfour-idcard:open', source, target, 'aviation')
